# Flightgear Phi web interface mod
---
A simple mod for the flightgear web interface, adding some basics extra features for the map (see description below).  
---
Creators:
* F-URAX - creator

  
Description :

Theses files are modification of original Phi/widgets interface (flightgear map in browser)

Location : flightgear folder / Contents / Resources / data / Phi / widgets
How to use :  simply make a backup of yours original files and replace them following filesystem

 
Modifications :


- Add cappability to switch between 2 baselayers : openstreetmap and opentopomap.  
	May be usefull when planing a flight to avoid relief collision especially in approach condition. 


- Add Openseamap overlay for fun and even (especially) for Carrier controled by player (and even seaplane).
	This allow navigation in compliance with marine traffic rules.
